# KinderOS Future Enhancements

This file tracks future product enhancements discussed during planning.

## HR, Payroll, and Staff Operations

- Payroll and salary management:
  - Salary structure per staff
  - Monthly payroll run
  - Deductions and adjustments
  - Payslips and export reports
  - Strict role access (OWNER, ADMIN, ACCOUNTANT)
- Leave and attendance automation:
  - Auto-mark attendance as leave for approved student leave requests
  - Multi-level leave approval workflows (optional)

## Parent and Communication

- Parent communication automation:
  - Push notifications
  - WhatsApp reminders
  - Fee due reminders
  - Homework and attendance alerts
  - Leave request decision alerts
- Parent portal upgrades:
  - Better homework filters (Today / This Week / Previous)
  - Event detail timeline improvements

## Academic and Student Progress

- Assessment and report cards:
  - Exam and assessment planner
  - Gradebook
  - Skill-based progress tracking
  - Printable report cards
- AI academic support:
  - Auto-drafted teacher comments
  - AI-assisted progress narratives

## Admissions and Growth

- Admission CRM enhancements:
  - Counselor assignment
  - Follow-up automation
  - Funnel and conversion analytics

## Operations and Finance

- Inventory and procurement:
  - Stock tracking for books and supplies
  - Vendor management
  - Purchase records
- Advanced analytics:
  - Fee collection trends
  - Attendance heatmaps
  - Class performance insights

## Transport and Safety

- Transport module expansion:
  - Route-level tracking
  - Pickup/drop attendance
  - Parent trip notifications

## Platform and Security

- Document vault:
  - Student/staff document repository
  - Expiry reminders
- Security and audit:
  - Action audit logs
  - Sensitive-data access logs
  - Backup and restore utilities

## Product Experience

  - Progressive Web App (PWA) support
  - Installable experience for Android, iOS, and desktop
